# 微信小程序接入通义千问 AI 指南

## 📋 准备工作

### 1. 获取通义千问 API Key

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 进入「API-KEY 管理」页面
4. 创建新的 API Key，复制保存（只显示一次）

### 2. 开通微信云开发

1. 打开微信开发者工具
2. 点击顶部工具栏「云开发」按钮
3. 开通云开发环境（免费版即可起步）
4. 记录环境 ID（如：cloud1-xxx-xxx）

- 🚀 便捷：免运维，按量计费

## 🔧 部署步骤

### 步骤 1：初始化云函数目录

```bash
# 在项目根目录创建云函数
mkdir -p cloudfunctions/qwenChat
cd cloudfunctions/qwenChat
npm init -y
npm install axios
```

### 步骤 2：配置云函数

在云函数目录创建 `config.js`：

```javascript
module.exports = {
  // 通义千问 API Key（从阿里云获取）
  DASHSCOPE_API_KEY: 'sk-xxxxxxxxxxxxxxxxxxxxxxxx',
  
  // 模型选择
  MODEL: 'qwen-turbo',  // 或 'qwen-max'（更聪明但更贵）
  
  // 系统提示词（设定 AI 人设）
  SYSTEM_PROMPT: `你是一位温柔耐心的高中数学老师，名叫"小 Q"。
你的学生是高中生，你需要：
1. 用亲切、鼓励的语气讲解数学题
2. 详细展示解题步骤，不要跳步
3. 多用比喻和生活化的例子帮助理解
4. 当学生答错时，先肯定再引导
5. 适当使用 emoji 让对话更生动
6. 专注高中数学：函数、三角、数列、不等式、立体几何、解析几何、概率统计、导数`
}
```

### 步骤 3：编写云函数代码

创建 `cloudfunctions/qwenChat/index.js`：

```javascript
const axios = require('axios')
const config = require('./config')

exports.main = async (event, context) => {
  const { question, conversationHistory = [] } = event
  
  // 构建消息
  const messages = [
    { role: 'system', content: config.SYSTEM_PROMPT },
    ...conversationHistory,
    { role: 'user', content: question }
  ]
  
  try {
    const response = await axios.post(
      'https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions',
      {
        model: config.MODEL,
        messages: messages,
        temperature: 0.7,
        max_tokens: 1000
      },
      {
        headers: {
          'Authorization': `Bearer ${config.DASHSCOPE_API_KEY}`,
          'Content-Type': 'application/json'
        },
        timeout: 30000
      }
    )
    
    return {
      success: true,
      answer: response.data.choices[0].message.content,
      model: config.MODEL,
      usage: response.data.usage
    }
    
  } catch (error) {
    console.error('AI 调用失败:', error.response?.data || error.message)
    return {
      success: false,
      error: error.response?.data?.message || error.message
    }
  }
}
```

### 步骤 4：配置 package.json

```json
{
  "name": "qwenChat",
  "version": "1.0.0",
  "description": "通义千问 AI 对话云函数",
  "main": "index.js",
  "dependencies": {
    "axios": "^1.6.0"
  }
}
```

### 步骤 5：上传云函数

在微信开发者工具中：
1. 右键点击 `cloudfunctions/qwenChat` 目录
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成（状态变为绿色）

### 步骤 6：修改前端代码

更新 `pages/chat/chat.js`，将模拟回复改为调用云函数（见代码注释）。

### 步骤 7：配置 app.js

在 `app.js` 中添加云环境配置：

```javascript
globalData: {
  cloudEnvId: 'cloud1-xxx-xxx',  // 替换为你的云环境 ID
  // ...其他配置
}
```

## 🔐 安全建议

1. **不要在前端代码中硬编码 API Key**
2. **云函数权限设置**：仅允许小程序调用
3. **内容审核**：可接入阿里云内容安全 API
4. **限流**：云函数中可添加调用频率限制

## ❓ 常见问题

### Q: 云函数调用超时怎么办？
A: 在 `config.js` 中增加 `timeout`，或在云函数控制台调整超时时间（最大 60 秒）

### Q: 如何切换模型？
A: 修改 `config.js` 中的 `MODEL` 字段，重新上传云函数

### Q: 小程序审核需要 AI 类目资质吗？
A: 是的，需要在小程序后台添加「人工智能」类目，可能需要算法备案

### Q: 如何实现流式输出（打字机效果）？
A: 需要使用云函数 WebSocket 或 SSE，参考微信官方文档
