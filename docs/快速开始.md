# 🚀 快速开始指南

> 5 分钟让你的小程序接入 AI！

---

## ⚡ 快速配置（3 步）

### 第 1 步：获取 API Key（2 分钟）

1. 访问 https://bailian.console.aliyun.com/
2. 登录阿里云账号（没有就注册一个）
3. 点击左侧「API-KEY 管理」
4. 点击「创建新的 API-KEY」
5. 复制生成的 Key（格式：`sk-xxxxxxxxxxxxxxxx`）

> 💡 新用户有免费额度，足够测试使用！

---

### 第 2 步：配置云函数（1 分钟）

1. 打开文件：`cloudfunctions/qwenChat/config.js`
2. 找到这一行：
   ```javascript
   DASHSCOPE_API_KEY: process.env.DASHSCOPE_API_KEY || 'sk-你的 API_KEY_在这里',
   ```
3. 替换为你的 API Key：
   ```javascript
   DASHSCOPE_API_KEY: 'sk-xxxxxxxxxxxxxxxxxxxxxxxx',
   ```
4. 保存文件

---

### 第 3 步：配置云环境（1 分钟）

1. 打开文件：`app.js`
2. 找到这一行：
   ```javascript
   cloudEnvId: 'cloud1-xxx-xxx',
   ```
3. 替换为你的云环境 ID（如果没有，看下面的「开通云开发」）

---

## 📦 开通云开发（如果还没开通）

1. 打开微信开发者工具
2. 点击顶部工具栏「云开发」按钮
3. 点击「开通云开发」
4. 选择「免费版」（个人使用足够）
5. 等待创建完成
6. 复制环境 ID（格式：`cloud1-xxx-xxx`）

---

## 📤 上传云函数

1. 在微信开发者工具中，右键点击 `cloudfunctions/qwenChat` 目录
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成（状态变为绿色✓）

> ⚠️ 如果看到错误，检查：
> - `config.js` 中的 API Key 是否正确
> - `package.json` 是否存在
> - 是否已开通云开发

---

## 🧪 测试

### 测试云函数

1. 点击顶部「云开发」按钮
2. 进入「云函数」→ 点击 `qwenChat`
3. 点击「测试」
4. 输入测试事件：
   ```json
   {
     "question": "什么是函数？",
     "conversationHistory": []
   }
   ```
5. 点击「确定」
6. 查看返回结果，应该看到 AI 回复

### 测试小程序

1. 点击顶部「编译」按钮
2. 进入「聊天」页面
3. 发送一条消息：「什么是函数？」
4. 等待 1-3 秒，应该收到 AI 回复

## ❌ 常见问题

### Q: 云函数上传失败
**A:** 确保在 `cloudfunctions/qwenChat` 目录下执行过 `npm install`

### Q: 调用返回「API 配置有问题」
**A:** 检查 `config.js` 中的 API Key 是否正确，确保没有多余空格

### Q: 调用超时
**A:** 在 `app.js` 中增加 `timeout` 值，或检查网络连接

### Q: 小程序提示「不在以下合法域名列表中」
**A:** 开发阶段可以在微信开发者工具中关闭「校验合法域名」：
   - 点击右上角「详情」
   - 勾选「不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书」
